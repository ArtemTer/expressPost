Артём С.Т. (серверная часть, оформление)

Описание проекта:
Сайт представляет из себя сервис для публикации постов на различные темы.
Есть система аккаунтов, комментариев и поиск по сайту.
Также сайт предоставляет удобный конструктор для создания постов и их оформления.

Технические характеристики:
Backend:
 Основной язык программирования: JavaScript (node.js)
 Язык программирования для работы с базами данных: SQLite
 Используемые библиотеки: express, argon2, sqlite3, body-parser, cookie-parser
 Тип архитектуры: микросервисный
 Сервис использует файлы Cookie

Frontend:
 HTML, css, ejs, JavaScript
 Используемые библиотеки: JQuery
